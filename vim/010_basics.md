## Was is Vim?

Vim ist ein Text-Editor für die so genannte Command-Line. 

---

## Modes

- Command Mode
	- Befehle ausführen
- Command Line Mode
	- Commands prefixed with `:`
- Insert Mode
	- Text/Code schreiben
- Visual Mode
	- Markieren

---

## Command Line Mode

| Command | Funktion                      |
| ------- | ----------------------------- |
| :w      | save                          |
| :q      | quit                          |


### Search
Search mode
```
/pattern
```
`n` drücken um zur nächsten Vorkommen des Suchbegriffs zu navigieren

---


### Search and Replace
```
:[range]s/{pattern}/{string}/[flags]
```
